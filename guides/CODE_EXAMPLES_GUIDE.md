## Overview

This guide teaches you to create code examples that are understandable, copy-correct, runnable, safe, and accessible. It is split into three chapters for easier navigation.

---

## Chapters

### [Chapter 1: Foundations](chapters/01-foundations.md)

Covers the essentials: syntax highlighting, code formatting, copy-paste readiness, self-contained examples, showing changes and output, annotations, accessibility, and minimal reproductions.

### [Chapter 2: Advanced Techniques](chapters/02-advanced.md)

Covers polyglot examples, versioning, collapsible code blocks, example security, live/interactive code, auto-generating examples from real code, license/attribution, and tool markup.

### [Chapter 3: Reference](chapters/03-reference.md)

Anti-patterns, practical exercise, cheat sheet, checklist, glossary, cross-references, known issues, and version history.
